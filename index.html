إعدادات الربط بالسيرفر

Supabase Cloud Database

كيف تحصل على المفاتيح:

1. ادخل على supabase.com وسوي حساب
2. أنشئ مشروع جديد (New Project)
3. انتظر 2 دقيقة لين يجهز المشروع
4. اذهب لـ Project Settings → API
5. انسخ Project URL (يبدأ بـ https://)
6. انسخ anon public key (طويل جداً)

مهم جداً:

  • لازم تسوي جداول (Tables) في Supabase قبل الاستخدام
  • الجداول المطلوبة: users, requests, inventory, transactions, inventory_history, alt_devices, alt_models, assignments
  • فعل RLS (Row Level Security) أو خله مطفي للتجربة
  • لو ما عندك سيرفر، اضغط "الاستمرار بدون سيرفر"

أنواع البيانات المطلوبة للجداول:

users: id(int8), username(text), password(text), full_name(text), role(text), created_at(timestamptz)
requests: id(int8), request_number(text), computer_number(text), device_name(text), employee_name(text), department(text), status(text), issue_description(text), procedures(jsonb), created_at(timestamptz), updated_at(timestamptz), created_by(text)
inventory: id(int8), name(text), qty(int4), max_qty(int4)
transactions: id(int8), item_id(int8), item_name(text), qty(int4), employee(text), date(text), damaged(text), notes(text), user(text)
inventory_history: id(int8), item_id(int8), item_name(text), type(text), qty(int4), old_qty(int4), new_qty(int4), date(text), user(text)
alt_devices: id(int8), type(text), total_qty(int4)
alt_models: id(int8), type(text), model(text), serial(text), status(text), location(text), added_at(timestamptz), updated_at(timestamptz), assigned_to(text), assigned_date(text)
assignments: id(int8), device_type(text), model(text), model_id(int8), employee(text), location(text), assign_date(text), return_date(text), status(text)
